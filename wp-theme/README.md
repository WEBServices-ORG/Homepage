# WEBServices WordPress Theme

This directory contains the WordPress theme version of the WEBServices website.

## Theme Structure

- `style.css` - Main stylesheet with theme header information
- `functions.php` - Theme functions and setup
- `index.php` - Main template file
- `header.php` - Header template
- `footer.php` - Footer template
- `screenshot.png` - Theme screenshot (880x660px recommended)

## Installation

### For WordPress.com (with GitHub Deployments)

1. Ensure you have a WordPress.com Business or eCommerce plan
2. Go to your WordPress.com site dashboard
3. Navigate to Code → GitHub Deployments
4. Connect your GitHub account
5. Select this repository
6. The theme will be automatically deployed when changes are pushed to the main branch

### For Self-Hosted WordPress

1. Download or clone this repository
2. Copy the `wp-theme` directory to your WordPress installation's `wp-content/themes/` directory
3. Rename the directory to `webservices` (optional)
4. Go to WordPress Admin → Appearance → Themes
5. Activate the WEBServices theme

## Customization

The theme can be customized through the WordPress Customizer:

- **Appearance → Customize → Hero Section**
  - Tagline: Customize the main tagline text
  - Domain Display: Set the domain text to display

## Features

- Clean, minimal dark theme design
- Responsive layout
- Custom header and footer
- Customizer integration
- HTML5 support
- SEO-friendly structure

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
