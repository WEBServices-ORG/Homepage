# WEBServices WordPress Theme

WordPress theme aligned 1:1 with the static homepage baseline.

## Included
- Minimal dark single-page layout
- Header + one-line hero
- Products card section
- Support icon row
- Legal footer links + disclaimer
- Local icon and favicon assets

## Structure
- `style.css` - Theme styles + theme metadata
- `functions.php` - Theme setup and stylesheet enqueue
- `index.php` - Main page layout
- `header.php` - HTML head + favicon
- `footer.php` - Footer and legal links
- `assets/icons/` - Brand icon files
- `assets/favicons/` - Favicon files

## Installation
### WordPress.com (GitHub Deployments)
1. Open site dashboard
2. Go to Code -> GitHub Deployments
3. Connect repository and branch
4. Deploy `wp-theme` directory

### Self-hosted WordPress
1. Copy `wp-theme` to `wp-content/themes/webservices`
2. Activate in Appearance -> Themes

## Requirements
- WordPress 5.0+
- PHP 7.4+
