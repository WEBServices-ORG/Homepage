Lemon Squeezy API Reference

Store: https://store.webservicesdev.com

Overview

Lemon Squeezy API is REST-based with predictable resource URLs, JSON:API responses, and standard HTTP methods/status codes.

Test Mode

Use Test mode API keys for development and integration testing. Move to Live mode keys for production deployments.

Versioning

Endpoints include major version prefixes, for example /v1. Backward-incompatible changes are introduced in new major versions.

Rate Limits

  • Limit: 300 API calls per minute
  • Headers: X-Ratelimit-Limit, X-Ratelimit-Remaining
  • Exceeded limit response: 429 Too Many Requests
